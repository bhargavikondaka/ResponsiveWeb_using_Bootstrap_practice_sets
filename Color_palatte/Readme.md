# 🎨 Responsive Color Palette

A simple beginner-friendly web project built using **HTML, CSS, and Bootstrap 4**. This project demonstrates Bootstrap's responsive grid system and contextual color classes by displaying colorful blocks that automatically adjust their layout across different screen sizes.

## 🚀 Features

- Responsive layout using Bootstrap Grid
- Bootstrap contextual color classes
- Mobile-first design
- Clean and simple user interface
- Responsive across different screen sizes

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4

## 📂 Project Structure

```text
Responsive-Color-Palette/
├── index.html
├── style.css
└── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/responsive-color-palette.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.
