# 📱 Responsive Boxes Layout

A simple beginner-friendly web project built using **HTML, CSS, and Bootstrap 4** to demonstrate the Bootstrap Grid System and responsive layouts.

## 🚀 Features

- Responsive grid layout
- Bootstrap 4 Grid System
- Different column sizes for various screen widths
- Clean and simple UI
- Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4

## 📂 Project Structure

```text
Responsive-Boxes/
├── index.html
├── style.css
└── README.md
```

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/responsive-boxes.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.
